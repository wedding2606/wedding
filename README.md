### 2. Настройка отправки email через EmailJS

**Шаг 1: Регистрация**
1. Перейдите на https://www.emailjs.com/
2. Создайте бесплатный аккаунт (200 писем/месяц)

**Шаг 2: Создание Email Service**
1. Email Services → Add New Service
2. Выберите Gmail (или другой)
3. Подключите свою почту
4. Скопируйте **Service ID** (например: `service_abc123`)

**Шаг 3: Создание Template**
1. Email Templates → Create New Template
2. **Subject**: `Новый ответ на свадебное приглашение от {{from_name}}`
3. **Content**:
```
Получен новый ответ на приглашение!

Имя: {{from_name}}
Присутствие: {{attending}}
Количество гостей: {{guest_count}}
Предпочтения по алкоголю: {{alcohol_preferences}}
Аллергии: {{allergies}}

---
{{message}}
```
4. **To Email**: `forcursor_100@mail.ru`
5. Скопируйте **Template ID** (например: `template_xyz789`)

**Шаг 4: Получение Public Key**
1. Account → General → API Keys
2. Скопируйте **Public Key** (например: `user_aBcDeFgHiJkLmNoPqR`)

**Шаг 5: Обновление кода**

Найдите в файле **строку 1239**:
```javascript
emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', emailData, 'YOUR_PUBLIC_KEY')
```

Замените на:
```javascript
emailjs.send('service_abc123', 'template_xyz789', emailData, 'user_aBcDeFgHiJkLmNoPqR')
```

Найдите **строку 1254**:
```javascript
// emailjs.init('YOUR_PUBLIC_KEY');
```

Раскомментируйте и замените:
```javascript
emailjs.init('user_aBcDeFgHiJkLmNoPqR');
```

## 🐛 Решение проблем:

### Музыка не воспроизводится
- Современные браузеры блокируют автовоспроизведение
- Гость должен нажать на кнопку плеера
- Убедитесь, что путь к MP3 файлу правильный

### Форма не отправляется
- Проверьте ключи EmailJS (Service ID, Template ID, Public Key)
- Откройте консоль браузера (F12) для просмотра ошибок
- Если EmailJS не работает, откроется mailto (почтовый клиент)

### Сердечко не двигается в таймлайне
- Убедитесь, что JavaScript включен
- Попробуйте обновить страницу
- Проверьте консоль на ошибки
